What is RedSide?

RedSide is the secure Cloud storage app.

Data transmitted across networks that you don't control, or stored on devices or in the Cloud on systems not under your control can be assumed to be insecure. Most Cloud Storage providers offer "secure" storage, but the data is not encrypted until it is already in the Cloud. This means that such providers can decrypt the data, because they hold the encryption keys. Or the data is only secured during transportation. This has been demonstrated repeatedly in the recent past to offer security that is often easily breakable and susceptible to serious software security bugs.

If you want secure data, it must be encrypted on devices under your immediate control and using keys to which only you have access. Anything less is only apparent security.

What does the name RedSide mean?

The name RedSide is taken from communications technology in which data that is transmitted across insecure networks or stored in insecure storage is encrypted. The data is only available in unencrypted form on the red side of such devices.

How does the RedSide App work?

RedSide offers data storage on your iPad and / or iPhone. There is a version available for OS X.

  • Upload data to and download data from remote server storage.
  • Synchronize unencrypted data with remote server storage. This is primarily intended to synchronize your data with your local computer, which is fully under your control. (It is not recommended to synchronize unencrypted data with Cloud Storage.)
  • Synchronize encrypted data with remote server storage. The data is anonomized, compressed and encrypted on your local device before transmitting to the Cloud storage.
  • Synchronize multiple devices (iPads and iPhones) using the encrypted Cloud storage.

Which Cloud Servers does RedSide support?

RedSide Version 1.0.0 supports Dropbox© and SFTP (OS X, Linux, and UNIX servers.)
